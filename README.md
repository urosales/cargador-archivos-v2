# Cargador de archivos v2

Un componente ligero en **HTML, CSS y JavaScript** para subir archivos mediante **drag & drop** o seleccionándolos desde el explorador de archivos.

No requiere dependencias ni frameworks.  
Simplemente configuras  los parametros e instancias el componente

## Características

- Arrastrar archivos a una zona (drag & drop)
- Seleccionar archivos con botón
- Los archivos se suben en cuanto se depositen en la caja
- Puedes configurar que automáticamente o manualmente se eliminen de la caja una vez se hubieran cargado a un servidor
- Sin dependencias
- Fácil de integrar en cualquier proyecto
